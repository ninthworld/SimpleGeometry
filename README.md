# SimpleGeometry
3D Voxel Engine using LWJGL
* GreedyMeshing voxels along all axis for each chunk
* 2D SimplexNoise terrain generation
* Simple SSAO shader
* Simple face lighting
* Simple shadow mapping
* Loading voxel models
